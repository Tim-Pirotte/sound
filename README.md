# Sound
This is a repository that I use to learn embedded programming. It makes a piezo buzzer play RTTTL songs using PWM on a Pi Pico 2W.

These are the sources from which I try to apply concepts:
    - Modern Embedded Systems Programming Course by Miro Samek (https://youtube.com/playlist?list=PLPW8O6W-1chwyTzI3BHwBLbGQoPFxPAPM&si=M_IQ8LVcg4YicJIH)
    - Embedded Linux and FPGAs by Nitin Chandrachoodan (https://youtube.com/playlist?list=PLUQpHm_JtukJ5tUmlcWKpcVbGfYy0MfxI&si=4Gh8amg5Hu80EhTk)

Since I am trying to apply all concepts some parts of this codebase would be different if they weren't for learning. Take for example polymorphism for selecting the parser when a command comes in. I would probably just use a switch for this instead.
